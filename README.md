## ✨ Overview

**Tibeb** is more than an eBook reader.

It is a modern reading companion designed to transform books into lasting knowledge. Whether you're reading an EPUB, studying a PDF, or listening to an audiobook, Tibeb keeps your entire reading journey in one place—from your library and reading progress to achievements and personal goals.

Our vision is simple:

> **Transform every page you read into lasting wisdom.**

## 🧠 Design Philosophy

Tibeb’s gamification system is inspired by modern behavioral design principles:

- Progress must always be visible
- Small actions must feel meaningful
- Consistency is more valuable than intensity
- Motivation is built through feedback loops, not pressure
- Learning should feel rewarding, not forced

Every interaction is designed to help readers return tomorrow — not just finish today.

# 🤝 Contributing

Contributions of all sizes are welcome.

Whether you're fixing a bug, improving documentation, refining the UI, or building new features, your contributions help make Tibeb better for everyone.

1. Fork the repository
2. Create a feature branch

```bash
git checkout -b feature/amazing-feature
```

3. Commit your changes

```bash
git commit -m "feat: add amazing feature"
```

4. Push your branch

```bash
git push origin feature/amazing-feature
```

5. Open a Pull Request

Before contributing, please check existing issues or open a discussion for larger changes.
